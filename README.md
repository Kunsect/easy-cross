# easy-cross

> 基于 EasyXDM 实现第三方 JsSDK 的跨域通信的解决方案


## 安装

1.克隆本库

 ```
  git clone github.com/Kunsect/easy-cross.git
 ```
 
## 配置

1. 修改 hosts 文件

```
  127.0.0.1  www.into.dev
  127.0.0.1  www.third.dev  
```

2. 配置本地域名
